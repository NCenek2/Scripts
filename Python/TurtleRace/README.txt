Welcome to the Turtle Race Folder. You can view the code 
used to create Turtle Race or you can download the executable to 
play on a Windows device.

Controls:
WASD and Space Bar.

You only need to click once to make the turtle move in a specific 
direction. Space bar will direct the turtle to the center.

The opponent turtle is slightly faster, but if you are close together,
the user has higher priority to score. 

First to 10 Wins! To exit after someone wins, just click the screen.
Enjoy!
